Neuronal Spike Pattern Analysis

Signal Processing and Analysis for Alzheimer's Research

View MATLAB Code View Python Code

Project Overview

At the Byrd's Alzheimer Institute, I was responsible for analyzing electrical spike patterns from neuronal networks using multielectrode arrays (MEA). This project involved a complete workflow, from conducting wet lab procedures for cell culturing to developing custom analysis scripts. The goal was to process complex raw data into meaningful insights about neuronal activity, contributing to research on Alzheimer's disease.

Project Breakdown

Wet Lab & Cell Culture

Performed essential wet lab procedures, including maintaining and passaging HEK cells and conducting primary neuron cultures to ensure high-quality samples for analysis.

Data Acquisition with MEA

Utilized multielectrode arrays (MEA) and a multichannel analyzer to record raw electrical signals from neuronal networks, sampling at 25kHz per channel.

MATLAB & Python Scripting

Developed custom scripts in both MATLAB and Python to automate the data analysis pipeline, increasing data processing efficiency by 30%.

Frequency Analysis

Applied Fourier transforms to perform frequency analysis on the signals, identifying power in different EEG bands and detecting spike trains using custom filters.
